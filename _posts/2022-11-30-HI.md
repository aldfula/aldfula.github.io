## git

- 오픈 소스 버전 관리 시스템
- 로컬에서 버전 관리
- 소프트웨어 개발 및 소스 코드 관리에 사용

## github


- git repository를 위한 웹 기반 호스팅 서비스
- 클라우드 서버를 사용해 로컬에서 관리한 소스를 업로드해 공유 가능
- 분산 버전 제어, 엑세스 제어, 소스 코드 관리 등을 제공

## git과 github의 차이

- git은 버전 관리 프로그램
- github는 원격 저장소
